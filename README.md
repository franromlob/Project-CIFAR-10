# AI Model Export & Management (TensorFlow)

## 📌 Overview
This project focuses on the **Exportation for Production** of Deep Learning models using **TensorFlow**. It demonstrates the workflow of transitioning from experimental training in Jupyter Notebooks to structured, deployable model formats.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** TensorFlow / Keras
* **Environment:** Virtual Environments (venv)
* **Version Control:** Git & GitHub

## 🚀 Key Features
* **Model Serialization:** Implementation of `tf.saved_model.save` for cross-platform compatibility.
* **Production Readiness:** Transitioning models from `.ipynb` to the standard **SavedModel** format.
* **Architecture:** Designed with a focus on **Scalability** and modularity for AI solutions.

## 📂 Project Structure
* `SavedModel.py`: Core script for model exportation.
* `.gitignore`: Configured to exclude heavy datasets and virtual environments to maintain a clean repository.
* `requirements.txt`: List of dependencies for reproducibility.
